# React Mini Projects

A collection of **mini React applications** demonstrating practical React skills, modern UI patterns, and real-world frontend development. This repository is continuously growing as new projects are added.  

Each project is **standalone**, easy to run, and ideal for learning or building a portfolio.

---

## 📁 Projects Included

| Project Name          | Description                                           | Folder |
|----------------------|-------------------------------------------------------|--------|
| **Landing Page**      | A modern, responsive landing page with navigation, hero sections, and call-to-action buttons. | `LandingPage` |
| **CurrencyConvertor** | Convert between multiple currencies with live calculations. | `CurrencyConvertor` |
| **ThemeSwitcher**     | Toggle between light and dark themes using React state. | `ThemeSwitcher` |
| **bgChanger**         | Change background colors dynamically with user input. | `bgChanger` |
| **passwordGenerator** | Generate secure passwords with customizable length and characters. | `passwordGenerator` |
